Scripts and summary files for enrichemnt analyses in DDD and SSC

File descriptions:
-ddd_xchr_dnms.csv: DNMs for X Chr subset from DDD

-dnm.lof.validation.txt: Full DNM data for DDD and SSC

-forZach_synexp_extended_denovoWEST_results.txt: Provided by Kaitlyn Samocha. Contains expected number of mutations for each gene for synonymous and LOF mutations

-hs_plots.R: Contains scripts to plot main figures in manuscript, including enrichment analysis for DDD

-ssc_dnm.R: Contains scripts to plot figures and perform enrichment analysis in SSC
